# 🌸 Iris Flower Prediction App
### 📌 Overview

This is a simple Machine Learning web application built using Streamlit that predicts the species of an Iris flower based on its physical measurements. The model uses a trained classifier to predict whether a flower is Setosa, Versicolor, or Virginica.

### 🚀 Features
Interactive user interface using Streamlit
Real-time prediction based on user input
Probability scores for each flower species
Visual bar chart showing prediction confidence
Simple and user-friendly layout
🧠 Machine Learning Model

The application uses a pre-trained machine learning model (model.pkl) that:

Takes 4 features as input:
Sepal Length
Sepal Width
Petal Length
Petal Width
Outputs one of the three Iris species classes
### 🛠️ Technologies Used
Python 🐍
Streamlit 🎈
NumPy 📊
Scikit-learn (model training)
Pickle (model serialization)
